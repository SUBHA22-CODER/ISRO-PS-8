# Sample Outputs Description

This directory will contain pre-generated outputs from the POLARIS pipeline
for demonstration purposes. These files are generated by running:

```bash
python scripts/prepare_demo.py
```

## Expected Files After Demo Generation

| File | Description | Format |
|------|-------------|--------|
| `ice_probability.tif` | Ice probability map (0-1 per pixel) | GeoTIFF |
| `cpr_map.tif` | Circular Polarization Ratio map | GeoTIFF |
| `dop_map.tif` | Degree of Polarization map | GeoTIFF |
| `slope_map.tif` | Terrain slope in degrees | GeoTIFF |
| `psr_mask.tif` | Permanently Shadowed Region binary mask | GeoTIFF |
| `dsc_catalog.json` | Doubly Shadowed Crater catalog | JSON |
| `landing_sites.json` | Top-3 ranked landing sites with IRS scores | JSON |
| `rover_path_site_a.json` | Waypoint sequence for Site A traverse | JSON |
| `volume_estimate.json` | P10/P50/P90 ice volume estimates | JSON |
